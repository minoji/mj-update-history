== Description ==

with MJ Update History, plugins, theme updates and core updates can be saved in the log and displayed as a list or output in various formats.(example: csv)

== Installation ==

1. Upload `mj-update-history` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
